# Get Reviews

You can get a `List<Review>`, which represents a list of reviews of a single products using the `reviews()` method like this:

:::note
The `Review` object is a prebuilt class model that contains a single review informations such as reviewer's name, rating...
:::

#### Using `await`/`async` :

```dart
// we get by this all store's products
List<Product> allProducts = await YouCan.instance.products.all(); //

// we take the first one as example
Product singleProduct = allProducts.first;

// then we get it's reviews
// highlight-next-line
List<Review> singleProductReviews = await singleProduct.reviews();
```

#### Using A `FutureBuilder` :

```dart
FutureBuilder<List<Product>>(
    future: singleProduct.reviews(),
    builder: (context, snapshot) {
      if (snapshot.hasData) {
        return Text(snapshot.data);
      } else if (snapshot.hasError) {
        return Text("Error: ${snapshot.error}");
      } else {
        return CircularProgressIndicator();
      }
    },
  ),

```

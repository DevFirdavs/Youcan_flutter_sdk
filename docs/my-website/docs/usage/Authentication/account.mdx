# Account informations

After the user logged in in you store, you might want to get the his account details such as id, name, email, time of creation..., for this you will need to use the `customerAccount()` that will return a `CustomerAccountInformations` which contains all your users informations.

using `await`/`async`:

```dart
final customerInformations = await YouCan.instance.customers.customerAccount(
      userToken: "YOUR USER TOKEN",
      );
```
